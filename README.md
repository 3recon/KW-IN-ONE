# KW IN ONE

광운대학교 학생을 위한 크롬 확장 프로그램입니다.

학교를 다니면서 자주 확인하는 기능을 한곳에 모아, 더 빠르고 편하게 접근할 수 있도록 만들었습니다.  
광운대학교 관련 사이트 바로가기, 일정 관리, 공지사항 확인, 오늘의 학식 정보를 팝업 화면에서 바로 확인할 수 있습니다.

## 주요 기능

### 1. 바로가기
자주 사용하는 학교 관련 사이트를 빠르게 열 수 있습니다.

- KLAS
- 광운대 홈페이지
- E-러닝
- K-MOOC
- 웹메일
- 에브리타임
- 학식
- 교내전화번호
- 단과대 바로가기

### 2. 나의 일정
시험, 과제, 개인 일정을 추가하고 남은 시간을 확인할 수 있습니다.

- 일정 추가 및 수정
- 날짜/시간 선택
- 남은 시간 카운트다운 표시

### 3. 공지사항
광운대학교 공지사항을 카테고리별로 확인할 수 있습니다.

- 공지 목록 표시
- 카테고리 설정
- 필요한 공지만 빠르게 확인

### 4. 오늘의 학식
오늘의 학식 메뉴를 팝업에서 바로 확인할 수 있습니다.

- 메뉴 요약 표시
- 학식 페이지 바로가기 제공
